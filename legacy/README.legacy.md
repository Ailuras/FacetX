# Legacy (DocsBot v1)

This directory holds the **archived first version** of DocsBot: a Python
(`stdlib http.server`) backend with a SQLite-per-project data layer, a vanilla
JS/HTML/CSS single-page frontend, an MCP server, and a CLI.

It is **frozen** — no longer developed. DocsBot is being rebuilt as a native
macOS app that provides a project-oriented view over Apple Calendar and
Reminders (see `../REBUILD.md`).

The v1 code is kept here for reference. The quality fixes and pytest suite from
the final v1 commits live in git history.

Original docs: `README.md` in this folder.
